All smart contracts are here => https://github.com/Iqdar/Zombie/tree/master/src/contracts
Every function is explained in contract

To run this, you need to:
Install Ganache
Node.js
Metamask extension in google chrome
connect metamask with Ganache
Import at least one account from ganache by copying its private key

Open Node.js command prompt
1: cd /d (project path)
2: truffle test [tests your solidity code, on 0 passing, you are good to go]
3: truffle migrate --reset [deploys on blockchain]
4: npm run start [to run project]
